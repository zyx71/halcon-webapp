# Reflexión Final

Durante el desarrollo del análisis y diseño del sistema web para la empresa Halcón, pude reforzar mis conocimientos sobre los procesos empresariales y cómo se traducen en modelos técnicos. Utilizar herramientas como diagramas BPMN, diagramas de clases, diagramas de actividades y el modelo ER me permitió visualizar la arquitectura completa del sistema desde múltiples perspectivas.

Uno de los mayores aprendizajes fue entender cómo cada rol dentro de la empresa (ventas, compras, almacén y ruta) interactúa con el ciclo completo de una orden. Esto me ayudó a diseñar una base de datos sólida y funcional, considerando las relaciones clave y los procesos internos de la organización.

Además, practicar con herramientas como GitHub y MySQL Workbench fue fundamental para aplicar lo aprendido en un entorno más cercano al mundo laboral. Considero que este proyecto me permitió integrar teoría con práctica y entender la importancia de tener una visión clara y bien estructurada desde la fase de análisis.

Me siento satisfecho con los resultados obtenidos, ya que logré cumplir con cada uno de los entregables solicitados, organizando el flujo del sistema y proponiendo una solución coherente y escalable para el cliente. Sin duda, esta actividad me ayudó a crecer tanto académica como profesionalmente.
