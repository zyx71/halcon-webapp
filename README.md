# Halcon WebApp

Aplicación web desarrollada para la empresa Halcon, una distribuidora de materiales de construcción. El sistema permite a los clientes consultar el estado de sus pedidos, y a los empleados gestionar el ciclo completo de una orden desde que se coloca hasta su entrega.

---

## 🛠️ Metodología de trabajo

Se utilizó la metodología **Scrum** debido a su enfoque ágil e iterativo. Esta permite dividir el proyecto en Sprints y trabajar de manera organizada, adaptándose a cambios frecuentes y asegurando entregas incrementales.

### Roles en Scrum:
- **Product Owner:** Representa al cliente y define las prioridades.
- **Scrum Master:** Facilita el proceso y remueve obstáculos.
- **Development Team:** Diseña, desarrolla y prueba la aplicación.

### Ciclo de trabajo propuesto:
1. Sprint 0 - Análisis y diseño
2. Sprint 1 - Autenticación y panel administrativo
3. Sprint 2 - Módulo de pedidos
4. Sprint 3 - Consulta de clientes y carga de fotos
5. Sprint final - Pruebas y entrega
